# CodeAlpha_Network_Sniffer

This project is a functional network packet sniffer developed as part of my Cybersecurity internship with CodeAlpha. The tool is designed to intercept and log network traffic, providing insights into data flow, protocols, and packet structures.

## 🛠 Features
- **Real-time Capture**: Intercepts packets traversing the network interface.
- **Protocol Analysis**: Identifies and categorizes traffic into TCP and UDP protocols.
- **Data Inspection**: Extracts and displays source/destination IP addresses and raw payloads.
- **Educational Insight**: Demonstrates the mechanics of network communication and the visibility of unencrypted data.

## 📋 Requirements
- Operating System: Kali Linux (Recommended)
- Language: Python 3
- Library: `scapy`

## 🚀 How to Run
1. Clone this repository: `git clone https://github.com/YOUR_USERNAME/CodeAlpha_Network_Sniffer.git`
2. Navigate to the directory: `cd CodeAlpha_Network_Sniffer`
3. Ensure dependencies are installed: `pip install scapy`
4. Run the script with root privileges: `sudo python3 sniffer.py`

## 🛡 Ethical Disclaimer
This tool is for educational purposes only. Unauthorized sniffing of network traffic is illegal and unethical. Ensure you have explicit permission before monitoring any network.
